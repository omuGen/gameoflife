# omuGen/gameoflife

A rough and badly optimized implementation of Conway's Game of Life in C++ using SDL2 and CMake (by a beginner in all three).

### Controls

'space' to pause the game.\
's' to slow down.\
'f' to run faster.\
'r' to reset to random board.
